Bubble Sort
------------------
Statement : You are given a list of integers. Your task is to sort the list in ascending order using
the Bubble Sort algorithm
--------------------------------

Example 1:

Input: 5 2 9 1 5
Output : 1 2 5 5 9

-------------------
Example 2 :
Input : 10 3 8 4 2 7
Output : 2 3 4 7 8 10

Note : Stop early if the array is already sorted ( no elements are swapped)
